About Micro Service

Micro Services is an architectural style in which applications are decomposed into loosely coupled services.With fine-grained services and lightweight protocols

Advantages
  • Increased resilience
  • Improved scalability
  • The ability to use the right tool for the right task
  • Faster time to market
  • Continuous delivery
disadvantages
  • Robust monitoring is a must
  • You must embrace devops culture
  • Testing can be complex
  • You need to design with failure in mind
  • Continuous delivery

How to design your spring boot REST micro services

  • Follow Symantic Versioning for your project
  • Best Api End Point Descriptions
  • Use understandable variable names
  • Use Meaningful werbs
  • Use SSL/TLS certificates
  • Use access token filter

Micro Service offers

  • increased modularity
  • making applications easier to develop
  • making applications easier to test
  • making applications easier to deploy
  • easily changeable and maintainable
